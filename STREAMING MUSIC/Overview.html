Overview


What are Streaming music websites?

  • Basically, Streaming music websites are internet platforms where people can play songs on it.

    • An user can search for any song through a filter. In a filter you can enter only these things that the song should have. A filter includes characteristics like genre, artist, album and realeas-year.


    What are the goals of Streaming music websites?

    The goal of each Streaming music website is to play songs. People whith the same filters can make a playlist of songs. When someone found a playlist another user can hear it to. When all this went well and the user can like his playlist or album.


    How do Streaming music websites work?

  • The first step for the user is to create a new profile. This is very important, bacause in the profile are all informations of the user. If some informations are wrong, you will get wrong songs as suggestion, beacause the filter system searches only for songs that are similar to your profile.

    • The profile contains informations such as age, interests, which artist you prefer and so on. After the profile is correctly filled out, the user will get suggestions of other songs/playlists or the user can search for other songs or playlists.
